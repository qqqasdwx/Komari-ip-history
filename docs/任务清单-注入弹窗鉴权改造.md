## 1. 当前目标

这一轮只做以下主线，不扩别的功能：

1. 接入 `Komari` 身份判断
2. 拆游客只读公开接口
3. 拆详情接口与历史接口
4. 按新状态机收口 loader / `connect` / 回跳

不在本轮处理的内容：

- 继续抠样式细节
- legacy 全量删除
- loader 性能优化
- 历史能力全链路下线
