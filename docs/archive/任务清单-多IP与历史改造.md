更新时间：2026-04-02

---

## 1. 本轮目标

这一轮只做以下主线：

1. 节点下支持多个手动维护的目标 IP
2. 每个 IP 独立保存最近一次结果与历史
3. 上报改为节点级 token + 单 IP 单次上报
4. 节点详情页改为 IP 标签页模型
5. 历史改为按选中 IP 查看

不在本轮一次性做死的内容：

- 复杂拖拽交互的细修
- 历史差异高亮与趋势分析
- 把所有旧数据自动完美迁移成多 IP 结构
