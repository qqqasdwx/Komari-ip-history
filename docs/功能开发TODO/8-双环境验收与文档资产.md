# 第八步：双环境验收与文档资产

## 目标

为上述功能建立可重复的本地验收环境和文档资产，覆盖默认 Komari 主题与 PurCarte 主题，减少后续每个功能分支的验收成本。

- 两套环境的数据目录隔离。
- 两套环境的 IPQ 后端、前端、Komari 入口互不串扰。
- 提供一键停止和状态查看脚本。

## 不包含

- 不新增产品功能。
- 不替代单元测试。
- 不把本地双环境作为生产部署方式。

## 验收清单

- 一键启动后能访问默认 Komari、默认 IPQ、PurCarte Komari、PurCarte IPQ。
- 两套环境均可完成 Komari 节点创建、IPQ 接入、配置、查看结果。
- Playwright 脚本能在两套环境中跑完核心流程。
- 截图不是空白、加载态、错误页或骨架屏。
- 文档截图与当前 UI 一致。
- 停止脚本能清理本地预览进程。
- 验收文档能明确标注通过、失败和待处理项。
