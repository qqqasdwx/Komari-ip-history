## 1. 本轮目标

这一轮只做以下主线：

1. 节点详情页只主看当前结果
2. 历史进入独立页面查看
3. 历史按目标 IP 独立切换
4. 历史页支持快照对比

不在本轮一次性做死的内容：

- 图表化趋势分析
- 自动生成复杂变化结论
- 游客历史公开页
