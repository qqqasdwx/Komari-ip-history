## 1. 本轮目标

这一轮真正要完成的是：

1. 历史主页面改成字段变化列表
2. 历史页支持“所有 IP / 单个 IP”两种视角
3. 历史页支持时间范围与字段筛选
4. 快照对比页独立出来
5. 历史页与对比页统一使用展示页的翻译和颜色语义

不在本轮一次性做死的内容：

- 图表化趋势分析
- 自动生成复杂变化结论
- 游客历史公开页
